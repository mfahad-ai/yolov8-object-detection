# 🖼️ Object Detection using YOLOv8

## 📌 Overview
This project performs object detection on images using YOLOv8.

## ⚙️ Technologies Used
- Python
- YOLOv8
- OpenCV

## 🔍 Features
- Image input detection
- Bounding box prediction
- Object classification
